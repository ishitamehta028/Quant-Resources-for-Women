# Women in Quant — Resource Hub

A living, open-source collection of scholarships, fellowships, and funded opportunities for women (and anyone else who's been told this field "isn't really for people like them") trying to break into **quantitative finance** and **AI research**.

This repo exists because the best opportunities are almost never the ones that show up first in a Google search — they're scattered across university pages, foundation newsletters, and recruiter blog posts. This is an attempt to put the real ones in one place, kept current by an actual community instead of one person's bookmarks.

This repo is the companion to the [Newsletter Name] Substack — articles like *25 AI Research Fellowships You Should Actually Apply For* live there in long-form; the structured, trackable, contributable version lives here.

## A few ground rules for this repo

- **Every entry must link to an official source** — the program's own page, not a third-party aggregator. Scholarship/fellowship aggregator sites are frequently stale or wrong about deadlines and amounts.
- **Deadlines move.** Every table includes a "last verified" column. If you spot something stale, opening an issue or PR to fix it is one of the most useful contributions you can make — see the good-first-issue label.
- **This is for everyone**, even though the framing centers women specifically — these resources are open to anyone, and contributions from anyone are genuinely welcome.

## Contributing

Found a fellowship that should be here? Spotted a stale deadline? Want to add a personal note about your own application experience? Please do — see [`CONTRIBUTING.md`](CONTRIBUTING.md). First-time open-source contributors are explicitly, enthusiastically welcome; this repo is meant to be a good place to make your first PR.
