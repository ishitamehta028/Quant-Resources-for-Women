# Contributing to this repo

Thank you for even considering it — this repo is built on the idea that the best resource lists are maintained by a community, not one person. First-time open-source contributors are genuinely, especially welcome here. If you've never opened a pull request before, this is a good place to learn how.

## Easy ways to contribute (good first issues)

- **Add a resource.** Found a scholarship, fellowship, mentorship program, or community that should be in one of the resource files? Add a row, following the existing table format, with a link to the *official* source page.
- **Fix a stale deadline or broken link.** Programs change their cycles constantly. If you're applying to something in this repo and notice the date or amount is wrong, fixing it helps everyone who reads it after you.
- **Add a "last verified" date.** If you confirm an entry is still accurate, update its last-verified date — this is a tiny change that makes the whole repo more trustworthy.
- **Share your own experience.** Applied to something in this list? A short, honest note (what surprised you, what you wish you'd known) is incredibly valuable to the next person and very welcome as an addition to an entry.
- **Translate a resource file.** If you can confidently translate a resource list into another language, that's a meaningful, high-impact contribution.

## How to submit a change

1. Fork this repo.
2. Make your change on a new branch.
3. Open a pull request with a short description of what you added or fixed.
4. If you're adding a new resource, please include the official source link and the date you verified the information.

That's it. No change is too small. A one-line fix to a deadline is just as welcome as a brand-new resource file.

## Code of conduct, in short

Be kind. Assume good faith. This repo exists partly to make this field feel more reachable to people who've been told it isn't for them — let's keep that energy in every interaction here.

## Questions?

Open an issue, or start a discussion. There's no such thing as a question too basic to ask here.
